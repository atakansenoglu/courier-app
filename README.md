## Description

CRUD app for courier location tracking.

## Installation

Create .env file and insert your remote database credentials.
You can use MongoDB Atlas for that matter.

```bash
$ npm install
```

## Running the app

```bash
$ docker-compose up
```
```bash
# development
$ npm run start
```
## Swagger

```bash
You can use localhost:3000/api to use Swagger API.
```
